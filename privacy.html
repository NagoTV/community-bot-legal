Privacy Policy

# Privacy Policy **Last Updated:** June 2026 This Privacy Policy explains how the Discord bot ("the Bot") collects, uses, and stores data. By using the Bot, you agree to this policy. --- ## 1. Data Collected The Bot may collect and store the following types of Discord-related data: * Discord User IDs * Discord Server (Guild) IDs * Channel IDs (for tickets, logs, events, and configuration) * Role IDs (verification, autorole, supporter roles, etc.) * Message content (only when required for moderation logs or ticket transcripts) * Moderation records (warnings, bans, mutes, cases) * Verification data (temporary captcha codes and expiration timestamps) * Ticket information (type, creator, messages within ticket channels if transcripted) * Event participation data (RSVP responses: attending, maybe, not attending) * Server configuration settings (stored via admin commands) * OAuth login identifiers (Discord ID when using the dashboard) We do not collect passwords, financial data, or private personal information beyond Discord-provided identifiers. --- ## 2. How Data Is Used Data is used strictly for Bot functionality, including: * Verification (captcha-based user authentication) * Moderation (warnings, bans, logs, case tracking) * Ticket system functionality and support workflows * Event management and participation tracking * Role management (autoroles and button role menus) * Server statistics tracking (members, online users, supporter counts) * Logging administrative and moderation actions * Website dashboard authentication via Discord OAuth --- ## 3. Data Storage Data is stored locally in a SQLite database on the Bot’s hosting environment. * Data is not sold * Data is not shared with third parties * Data is not used for advertising or profiling purposes --- ## 4. Data Retention Data is retained as long as the Bot is installed in a server. When the Bot is removed: * Server configuration data may be removed * Some moderation logs or tickets may remain unless manually deleted * Users may request deletion of stored data --- ## 5. Logging System The Bot may log: * Message edits and deletions * Member joins and leaves * Role updates * Moderation actions (warnings, bans, kicks, mutes) * Ticket creation and closure * Event participation updates These logs are used for moderation transparency and server management. --- ## 6. Website Integration If enabled, the Bot may connect to a web dashboard using Discord OAuth2 to: * Identify users securely via Discord * Link accounts for supporter role synchronization * Manage server configuration settings The Bot does not access or store Discord passwords or sensitive authentication data. --- ## 7. Security Reasonable security measures are used to protect stored data. However: * No system is completely secure * We cannot guarantee absolute protection against data breaches or data loss --- ## 8. User Rights Server members and administrators may request: * Access to their stored data * Correction of data * Deletion of data (where technically possible) Requests should be made to the server owner or bot administrator. --- ## 9. Third-Party Services The Bot may interact with: * Discord API (required for functionality) * External web dashboard (if enabled) No third-party analytics or advertising services are used. --- ## 10. Changes to This Policy This Privacy Policy may be updated at any time. Continued use of the Bot implies acceptance of any changes. --- ## 11. Contact For privacy concerns, contact the server administrator or bot owner.
